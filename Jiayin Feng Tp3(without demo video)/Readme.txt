Name of the term project: Escape!
Short description: This project is a 3D raycasting in a mazes. The player starts on the left top corner of a maze and with an aim to reach the right bottom of the 2D Maze in the limited time.
How to run the project: put the 'TP3 codes' file in the same folder as 'cmu_cs3_graphics' and run this file.
Library used: cmu_cs3_graphics
Shortcut: 2D hint and maze hint buttom for testing 2D maze; And easy mode for a easier test case ( smaller maze, easier to get through).
